How to automate the compiler:
 - create your main.fop with the code you like on raspi
 - first run: do a POST request on /compile with code=...&mode=setup for setup and pipe it to bash
 - every next run: do a POST request on /compile with code=...&mode=main and store it to ~/raspi/main.py on your raspi, then restart ~/raspi/main.py
 - screen -r -d to monitor all values (Ctrl+A, wait, D for detatching from screen session)
